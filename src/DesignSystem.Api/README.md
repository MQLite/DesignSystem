# DesignSystem (PoC) – SQLite Skeleton (VS Code)

## Prerequisites
- .NET SDK 8.x
- VS Code + C# Dev Kit

## Run
From repo root:

```bash
dotnet restore
dotnet run --project src/DesignSystem.Api
```

Open Swagger (Development):
- https://localhost:5001/swagger (port may vary)

## SQLite
- Stored at ./data/app.db
